Database and SQL Course

I had read a lot about backend data management before this beginning this course so it was awesome to finally see work with data. We used MySQL with SSMS for most of this course. I practiced a lot of queries using the large AdventureWorks2017 database and practice building and modifying my own mini databases. This course was really enjoyable and a big breakthrough for my overall understanding of full-stack software development.

A few of the topics covered in this course:

  • Database Fundamentals
  • Importance of Databases
  • Crud Operations
  • RDBMS
  • Databases and Web Apps
  • Creating your own DB
  • MySQL
  • Stored Procedures
  • Using MySQL Server
