Overview of Software Development Course

This course began to go into the fundamental ideas of developing software. I learned a lot about Object-oriented programming course. I was also introduced to the different tools a software developer has at their disposal when creating software.

A few of the topics covered in this course:

  • Object-oriented Programming Basics
  • Web Application Basics
  • Database Basics
  • How programs are made
  • Thinking Like A Programmer
  • Number Systems
  • Data Structures
  • Basic security Concepts
