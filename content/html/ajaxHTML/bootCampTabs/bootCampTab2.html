Computer and Technology Basics Course

This course taught me a lot about basic definitions and concepts related to technology and computers. A lot of this course was a review for me, but other parts were brand new. My favorite part of this course was learning about different number systems and doing binary arithmetic.

A few of the topics covered in this course:

  • Clear definitions for every major technology term
  • Algorithm theory and design
  • Basic machine architecture
  • Central processing unit operation
  • Computer network principles
  • Internet design and operation
  • Web browser operation
  • Basic security concepts
